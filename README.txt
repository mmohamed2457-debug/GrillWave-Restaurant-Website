# GrillWave Restaurant Website

## Project Overview
GrillWave is a responsive restaurant website built using HTML, CSS, and JavaScript.

The project simulates a real online restaurant system with:
- User Authentication
- Admin Dashboard
- Dynamic Menu
- Shopping Cart
- Order Management
- Reservations
- Contact Messages
- Profile Management
- localStorage & Cookies

---

# Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- localStorage
- Cookies
- Responsive Design

---

# Main Features

## User Features
- Register new account
- Login system
- Dynamic food menu
- Search and filter menu items
- Add items to cart
- Update cart quantity
- Save orders
- Book reservations
- Save profile data
- Send contact messages

---

## Admin Features
- Admin login
- View all users
- View reservations
- View orders
- Delete orders
- Add menu items
- Delete menu items
- Full dashboard access
